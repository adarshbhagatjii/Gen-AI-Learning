## 🔹 2. Natural Language Processing (NLP)
NLP enables machines to process, understand, and generate human language.

---

## 🔹 3. NLP Pipeline
### 📌 1. Tokenization
Breaking text into smaller units:
- Sentence Level
- Word Level
- Character Level  
**Libraries Used:** NLTK, SpaCy

### 📌 2. Text Preprocessing
- Text Cleaning  
- Stopword Removal  

### 📌 3. Normalization
- Stemming  
- Lemmatization  
- Part-of-Speech (POS) Tagging  

### 📌 4. Text Vectorization
- One-Hot Encoding (OHE)  
- Bag of Words (BoW)  
- TF-IDF  

### 📌 5. Word Embeddings
- Dense vector representations  
- Captures semantic relationships between words  

---

## 🔹 4. Large Language Models (LLMs)
### 📌 Architecture
- Transformer-based models  
- Pretrained on large datasets  

### 📌 Components
- Input Prompt  
- Model Processing  
- Generated Output  

### 📌 Key Concepts
- Context understanding  
- Text generation  
- Few-shot & Zero-shot learning  

---

## 🔹 5. Prompt Engineering
### 📌 Key Concepts
- Designing effective prompts  
- Zero-shot, One-shot, Few-shot prompting  
- Controlling output format and quality  

---

## 🔹 6. LangChain
LangChain helps in building applications powered by LLMs.

### 📌 Components
- Models → LLM APIs  
- Prompt Templates → Structured input  
- Output Parsers → Structured outputs (JSON, etc.)  
- Chains → Sequential workflows  
- Memory → Context retention  
- Agents → Tool-based reasoning systems  

---

## 🔹 7. RAG (Retrieval-Augmented Generation)
### 📌 Architecture
Retriever → Context → LLM → Output  

### 📌 Components
- Vector Database  
- Embeddings  
- Retriever  

### 📌 Key Concepts
- External knowledge integration  
- Reducing hallucinations  
- Context-aware generation  

---

## 🔹 8. LangGraph
LangGraph enables building stateful, multi-step AI workflows.

### 📌 Architecture
- Graph-based system (Nodes + Edges)  
- State-driven execution  

### 📌 Pipeline
Input → State → Nodes → Conditional Edges → Output  

### 📌 Key Concepts
- Stateful workflows  
- Multi-step reasoning  
- Conditional branching  
- Advanced control over AI agents
